### 疯狂跳绳
使用HIIT方式进行跳绳锻炼的App

### 基本功能
* 使用HIIT（High-Intensity Interval Training）方法指导训练
* 支持设定热身时间和训练的轮次
* 支持简单的运动心率评估
* 支持语音播报运动进度，用的是iOS的语音合成技术
